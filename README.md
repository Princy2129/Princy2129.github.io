# DevHub - Modern Developer Platform

A production-ready website inspired by GitHub's design language built with Next.js, React, TypeScript, and Tailwind CSS.

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Open http://localhost:3000
```

## 📁 Project Structure

```
src/
├── app/              # Pages and layouts
├── components/
│   ├── common/      # Navbar, Footer, ThemeToggle
│   ├── sections/    # Hero, Features, Pricing, CTA
│   └── providers/   # ThemeProvider
└── styles/          # Global CSS
```

## ✨ Pages

- Home - Landing page
- About - Company info
- Features - Feature showcase
- Pricing - Pricing plans
- Contact - Contact form
- Login - Sign in
- Register - Sign up

## 🛠️ Tech Stack

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## 📦 Build

```bash
npm run build
npm start
```
